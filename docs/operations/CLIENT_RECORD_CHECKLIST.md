# Client record checklist

- [ ] Enquiry ID
- [ ] Client identity
- [ ] Service
- [ ] Trader address verified
- [ ] Exact official BTW ID verified
- [ ] Consultation confirmation version/date
- [ ] Invoices
- [ ] Offer version/date
- [ ] Terms version/date
- [ ] Withdrawal information and form sent
- [ ] Acceptance evidence
- [ ] Early-start request
- [ ] Approved third-party costs
- [ ] Approved scope changes
- [ ] Completion
- [ ] Complaints/cancellation/withdrawal
- [ ] Final CRM status
- [ ] Retention category
- [ ] Review/deletion date

File naming:

`YYYY-MM-DD_CLIENT_DOCUMENT-TYPE_VERSION`

Do not place passwords, unnecessary sensitive data, or full communication histories in CRM notes.

Use the smallest useful note set, and keep client records separate from any local working files.

# VANTAM Product Architecture V2

## Product Shape
VANTAM is a practical support service for international students in the Netherlands. The public site should show one clear entry point with a student-led message, then route visitors into the right service path.

## Public Offer Structure
### Consultation
- Relocation Orientation, €59
- Use when the visitor needs the first practical plan and wants help deciding the next step.

### Main Ladder
- VANTAM Start, €449
- VANTAM First Year, €899, recommended
- VANTAM Continue, returning-client-only in this cycle

### Housing Activation Path
- Housing Ready, €249
- Housing Search Campaign, €649

## Process Model
1. Assess
2. Map
3. Act & coordinate
4. Verify & continue

## Contact Model
- Keep the backend inquiry buckets unchanged.
- Prefill the contact form from the chosen public context.
- Use the public help identifiers to select the right meaning without changing storage shape.
- Show housing fields only when the selected help path is housing-related.

## Messaging Constraints
- No guarantee language.
- No response-time commitment.
- No portal/dashboard language.
- No legal-advice positioning.
- No redesign of routes or technical infrastructure.
